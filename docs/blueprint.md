# **App Name**: GSI Insight

## Core Features:

- Authentication & Profile: User registration/login with email/password, profile management including name, ID, major, level, campus, photo, contact info, and a scannable student QR code.
- Insight Dashboard: Personalized dashboard displaying next class, upcoming deadlines (assignments, payments), custom alerts, and a button to access the 'Ask Insight' AI assistant.
- AI Assistant - Ask Insight: A basic AI chat tool answering questions such as 'When is my next class?', 'What assignments are due?', and 'Where can I find a document?'
- Timetable: Displays a weekly/monthly calendar view of courses, automatic notifications for schedule changes, and reminders 15 minutes before each class.
- My Subjects: Access to the syllabus, learning materials (PDFs, videos, etc.), assignments, grades, and attendance records for each subject.
- GSI Library: Search for resources by major, semester, and subject, with offline downloading and the ability to mark items as favorites.
- Admin Services: Submit requests for transcripts or enrollment certificates and track the status of submissions (In progress / Approved / Denied)

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to convey trust and intellect.
- Background color: Very light gray-blue (#F0F4F8) provides a clean and calm background.
- Accent color: Violet (#7E57C2) for highlights, signifying creativity and inspiration.
- Body and headline font: 'Inter' (sans-serif) for a modern, machined, neutral look. Readable on various screen sizes.
- Consistent and clear icons for navigation and features.
- Clean and intuitive layout to help user flow and not be overwhelmed with information.
- Subtle transition animations on loading content or navigating to new pages.
